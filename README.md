# Create Backend Server

## Instructions

```bash
cd ~
cd ws
mkdir gomoku-backend
cd gomoku-backend
mkdir public
touch ./public/{index.html,index.js,index.css}
npm install -g http-server

```

## ./public/index.html
```bash
cat > ./public/index.html << 'EOF'
<!DOCTYPE html>
<html>
    <head>
        <link rel="stylesheet" href="/index.css">
    </head>
    <body>
        <h1>Hello World</h1>
        <script src='/index.js'/>
    </body>
</html>
EOF
```
